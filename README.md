# Shockwave News 

A news app project, 

## Description

This app has currently three main Features:

* Load news when page changed.
* Web view navigation.
* Simple and easy access to the latest articles.

## Tech Stack Used

* NewsAPI.
* HTTP requests.
* WebView.
